## 📁 Projects

### 1. 🧑‍💼 [Human Resources Dashboard](https://github.com/anuragpandey333/Human_Resources_DVA)
A dual-page HR analytics dashboard built to give a complete picture of an organisation's workforce. The Overview page covers headcount, attrition trends, department breakdown, gender & age demographics, education-performance mapping, salary by education & gender, and geographic location. The Details page provides a granular employee-level table with ID, role, location, salary, hire date, and tenure — all navigable via column filters.

**Tools:** Tableau, Power BI, Python, Pandas, NumPy, Matplotlib, Seaborn, SQL, Excel

---

### 2. ⚡ [EV Vehicle Population Analysis](https://github.com/anuragpandey333/EV_Vehicle_DVA)
An end-to-end interactive dashboard analysing the Washington State Electric Vehicle Population dataset with 150,000+ records. The project uncovers EV adoption trends over model years, geographic distribution across US states, brand dominance, CAFV eligibility breakdown, and model-level performance — all driven by dynamic filters for EV type, make, state, and CAFV eligibility.

**Tools:** Python, Pandas, NumPy, SQL, Power BI, Excel, Matplotlib, Seaborn, Tableau

---

### 3. 💳 [Loan Default Risk Analysis](https://github.com/anuragpandey333/SectionE_Group15)
An analytical project that identifies key risk factors driving loan defaults — examining borrower attributes such as credit score, income, loan amount, employment status, and repayment history to uncover patterns behind financial risk. Visualised through interactive dashboards, the project helps stakeholders assess default probability, segment high-risk borrowers, and make data-driven lending decisions.

**Tools:** Spreadsheet, Data Visualisation, Statistical Analysis

---

### 4. 🎵 [SpotifyPulse](https://github.com/anuragpandey333/E_G17_SpotifyPulse)
Analyzes 50K Spotify users to uncover churn & engagement insights via statistical analysis and an interactive Tableau dashboard.

**Tools:** Python, Pandas, Tableau, Matplotlib, Seaborn, Jupyter/Colab
